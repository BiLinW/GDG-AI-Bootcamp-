# 🎵 Playlist VibeCheck AI: Gemini-Powered Playlist Creator

This is a Flask web application that uses **Google Gemini 3 Preview** to translate a user's "vibe" into a curated Spotify playlist.

## 🚀 Features
- **AI-Powered Curation:** Uses natural language processing (Gemini) to understand complex mood descriptions.
- **Spotify Integration:** Automatically searches for tracks and creates a live playlist in the user's account via OAuth 2.0.
- **Real-time Feedback:** Responsive UI with loading states and direct Spotify redirects.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **AI Model:** Gemini 3 Preview 
- **API Wrapper:** Spotipy (Spotify Web API)
- **Frontend:** HTML5, CSS3 
 
## 📋 Prerequisites
Before running, ensure you have:
1. A [Google AI Studio API Key](https://aistudio.google.com/).
2. A [Spotify Developer App](https://developer.spotify.com/dashboard) (Client ID & Secret).
3. Python 3.9+ installed.
